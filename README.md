# Second Brain OS

WhatsApp voice AI agent for rural Karnataka farmers.

## What it does
Farmer speaks in Kannada on WhatsApp → transcribed via Groq Whisper → 
translated to English → matched against Karnataka government schemes → 
answer generated by Llama 3.3 70B → translated back to Kannada → 
replied on WhatsApp.

## Stack
Python, Flask, Groq API, Llama 3.3 70B, Twilio, Render, deep-translator

## RAG Pipeline
Uses sentence-transformer vector embeddings (all-MiniLM-L6-v2) for 
semantic retrieval — matches farmer queries by meaning, not keyword overlap.
